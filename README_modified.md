# 📌 Project Overview

This project focuses on predicting house prices using supervised machine learning regression techniques.

The project includes:

* ✅ Simple Linear Regression
* ✅ Multiple Linear Regression
* ✅ Polynomial Regression
* ✅ Gradient Descent Optimization
* ✅ Bias-Variance Analysis
* ✅ Model Evaluation Metrics

The objective is to analyze how different property features affect house prices and compare model performance.

# 📊 Dataset Features

| Feature                | Description               |
| ---------------------- | ------------------------- |
| `area_sqft`            | House area in square feet |
| `bedrooms`             | Number of bedrooms        |
| `bathrooms`            | Number of bathrooms       |
| `location_score`       | Location quality score    |
| `age_years`            | Age of property           |
| `distance_city_km`     | Distance from city        |
| `lot_size_sqft`        | Lot size                  |
| `has_garage`           | Garage availability       |
| `has_pool`             | Pool availability         |
| `renovation_years_ago` | Years since renovation    |
| `house_price_inr`      | Target variable           |

* Bias
* Variance
